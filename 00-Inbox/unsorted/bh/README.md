# REPORTS/

Weekly and monthly health reports generated by Body.Health.

---

## Naming Convention

```
YYYY-MM-DD_weekly.md
YYYY-MM-DD_monthly.md
YYYY-MM-DD_[topic].md
```

Examples:
```
2025-06-03_weekly.md
2025-06-01_monthly.md
2025-05-15_biomarker-review.md
2025-05-10_cardiovascular-audit.md
```

---

## Retention Policy

Keep all reports. They constitute the longitudinal health record.  
Do not delete or overwrite — append or create new files.
